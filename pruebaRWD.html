Diseño responsivo con CSS moderno

Este ejemplo muestra rejillas fluidas, unidades relativas, media queries, contenido adaptable e imágenes flexibles. Reduce o aumenta el tamaño de la ventana para ver cómo todo se reorganiza de forma suave.

Sección principal

A continuación, una rejilla de tarjetas que se adapta al ancho disponible gracias a auto-fit y minmax().

Rejillas fluidas

Las columnas se adaptan automáticamente, evitando barras de scroll horizontales y aprovechando mejor el espacio en pantallas grandes y pequeñas.

Unidades relativas

Usamos rem, %, vh y vw, haciendo que el diseño sea flexible cuando cambia el tamaño de fuente base o el tamaño de la ventana.

Media queries

A partir de ciertos anchos (768px, 1024px), reorganizamos la disposición de las áreas para mejorar la legibilidad y el uso del espacio.

Contenido adaptable

El texto, las imágenes y el layout se ajustan para conservar una buena experiencia de usuario en móvil, tablet y escritorio.

Otro bloque de contenido

Contenido lateral

Este aside cambia de posición según el ancho de la pantalla. En móvil se muestra debajo del contenido principal, y en pantallas más grandes aparece a la derecha como columna lateral.

Aquí podrías colocar un formulario corto, enlaces rápidos, un pequeño menú secundario, etc.
